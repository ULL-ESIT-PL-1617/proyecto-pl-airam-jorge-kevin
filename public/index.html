Descripción de uso del Lenguaje


  1. Las sentencias pueden ser asignaciones, funciones o declaraciones.

  2. Las funciones se declaran de la siguiente forma. Pueden ser declaradas en cualquier momento y accedidas globalmente:

    3. <type> ID(ID, ID, ...) { ... return ...; }

    Por ejemplo:

    void test(x){ x = 3; } numeric foo() { return 3; }

  3. Condicionales:

    4. if condicion { ... } if condition { ... } else if condition { ... } else if condition { ... } else { ... }

  4. Bucles:

    5. for (#1; #2; #3) { ... }[else {...}] while (parcondition) { ... }[else {...}] // #1 => Operaciones que se ejecutan antes de entrar al bucle. // #2 => Condición que se debe cumplir para que continue el bucle. // #3 => Operaciones que se ejecutan cada vez que se itera sobre el bucle.

    Por ejemplo:

    for (i = 0; i < 3; i = i + 1) { ... } else { ... } while (i < 50) { ... }

  5. La asignación puede se puede realizar a cualquier tipo de expresión Dichas asignaciones se declaran de la siguiente forma:

    6. y = 5; x = 3 * 2; z = foo( 3 * 4) * 4; h = 1 > 2;

    No es necesario declarar las variables previamente para que la asignación se produzca.

  6. Las condiciones toman valor true o false.

    7. Por ejemplo:

    condition1 = TRUE condition2 = i < 5

  7. Podemos declarar atributos de la clase de la siguiente forma:

    8. <visibility> <returnType> <name> = [<value> | <constructor>];

    Por ejemplo:

    public numeric x = 7; private otraClase y = otraClase.init();

  8. Para declarar métodos de una clase hacemos lo siguiente:

    9. <visibility> <functionDeclare> ;

    public void x() {} private numeric y(numeric a) { return a*7; }

  9. Para declarar una clase seguiremos la siguiente sintáxis:

    10. class <name> { [<atributos> | <metodos> ]* }

    Por ejemplo:

    class A { public numeric n = 0; } class B { public A a = A.init(); } class C { public B b = B.init(); }
